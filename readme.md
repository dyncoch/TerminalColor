# TerminalColor

Colors text in the terminal 

## Getting Started

### Prerequisites

```
Python 3
```

### Installing

Using the python Command

```
$ python3 vaccination_phase.py
```

Using the Script Filename

```
$ chmod +x vaccination_phase.py
$ ./vaccination_phase.py
```
